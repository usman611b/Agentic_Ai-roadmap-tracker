## 📌 What Is This?

This repository contains a **complete, structured learning system** for becoming an Agentic AI Engineer — the most in-demand AI specialisation in 2025.

It includes **three core resources**:

| Resource | Description |
|----------|-------------|
| 🗺️ **[Interactive Progress Tracker](./index.html)** | Web app with checkboxes, progress bars & milestones for all 180 days |
| 📘 **[Full Curriculum](./agentic_ai_curriculum.docx)** | 23-phase deep-dive curriculum with 100+ modules and 500+ topics |
| 📅 **[Daily Roadmap](./agentic_ai_daily_roadmap.docx)** | Day-by-day study plan with objectives, build tasks & deliverables |

## 🚀 Interactive Progress Tracker

**[👉 Open the Live Tracker](https://usman611b.github.io/ai-roadmap-tracker)**

The tracker is a fully interactive web app — no account needed, no install required.

### Features
- ✅ **Checkbox for every day** — mark days complete as you go
- 📦 **Sub-tasks per day** — 3 micro-tasks to tick off each day
- 📈 **Live progress bars** — per phase, per week, and overall
- ⭐ **26 milestone days** — highlighted and tracked separately
- 🔍 **Search & filter** — find any topic, tool, or deliverable instantly
- 💾 **Auto-saves progress** — stored in your browser, persists forever
- 🎉 **Confetti on completion** — celebrate every day you finish

## 🏁 How to Use This Repo

### Option 1 — Use the Live Tracker (Recommended)
1. Go to **[https://usman611b.github.io/ai-roadmap-tracker](https://usman611b.github.io/ai-roadmap-tracker)**
2. Start from Day 1 and check off tasks as you complete them
3. Your progress saves automatically in your browser

### Option 2 — Run Locally
1. Clone this repo:
   ```bash
   git clone https://github.com/usman611b/ai-roadmap-tracker.git
   ```
2. Open `index.html` in your browser — no server needed!

### Option 3 — Download the Docs
- Download `agentic_ai_curriculum.docx` for the full 23-phase curriculum
- Download `agentic_ai_daily_roadmap.docx` for the detailed daily plan

## 💡 Study Tips

- ⏱️ **4–6 hours/day** for the fast track (180 days)
- 🏗️ **Build every single day** — don't skip the Build Tasks

- 📓 **Keep a daily journal** — answer the reflection questions
- ⭐ **Celebrate milestones** — there are 26 of them!
